Comandos básicos de la terminal

pwd
Devuelve la ruta del directorio en que estás ubicado.
ls
Devuelve los archivos y carpetas que hay en el directorio en el que estás ubicado.
ls -al
Devuelve los archivos y carpetas que hay en el directorio en el que estás ubicado incluyendo los archivos ocultos.
ls -a
Muestra los archivos ocultos dentro de un directorio.
ls -l
Muestra los archivos en forma de lista en un directorio.
mkdir
Crea un nuevo directorio o directorios en la ubicación actual. Se debe agregar el nombre del directorio. Ejemplo: mkdir nombre-de-la-carpeta.
touch
Crea un nuevo archivo o archivos en la ubicación actual. Se debe agregar el nombre del archivo mas su extensión. Ejemplo: touch archivo.txt.
rm
Elimina archivos de un directorio. Se debe agregar el nombre del archivo y su extensión. rm archivo.txt.
rm -d
Elimina un directorio vacio. Se debe agregar el nombre del directorio. rm -d nombre-del-directorio-vacio.
rm -r
Elimina un directorio que contiene archivos. Se debe agregar el nombre del directorio. rm -r nombre-del-directorio.
rm -ri
Elimina un directorio que contiene archivos. Se debe agregar el nombre del directorio. rm -ri nombre-del-directorio. La terminal te preguntará si estas seguro de eliminar los archivos o carpetas y deberas seleccionar si o no.
head
Muestra las primeras 10 lineas de un archivo de texto.
head -r
Muestra las primeras de lineas que indiques de un archivo de texto.
tail
Muestra las últimas lineas que indiques de un archivo de texto.
tail -r
Muestra las ultimas 10 lineas de un archivo de texto.
less
Muestra el contenido de un archivo de texto.
cat
Muestra el contenido de un archivo especifico. Se debe agregar el nombre del archivo y su extensión. cat archivo.txt
cd
Permite ingresar a una capeta que este dentro del directorio. Se debe agregar el nombre de la carpeta. cd nombre-de-la-carpeta
cd ..
Permite volver a un nivel anterior de la carpeta en la que te encuentras.
history
Muestra el historial de todos los comandos que se han realizado.
! + número
Ejecuta el comando asignado a un número en el historial.
clear
Limpia la consola.
type
Muestra el tipo de comando.
man
Muestra información que el manual tiene del comando.
info
Muestra información general del comando.
whatis
Muestra una información bastante breve del comando.

Comandos básicos de git

git init
Inicia un repositorio en el directorio.
git status
Muestra el estado del repositorio.
git add .
Agrega los archivos del directorio al stage y los prepara para el primer commit
git commit -m
Agrega los archivos al repositorio dejando una version de los archivos en el historial
git commit -am
Permite agregar los cambios de un archivo o archivos en el repositorio y hacer un commit o version a la vez.
git log
Muestra todo el historial de los commits o versines realizadas de los archivos del directorio
git reset soft
Elimina los archivos del repositorio, pero no los elimina del stage ni del directorio.
git reset mixed
Elimina los archivos del repositorio y del stage pero no del directorio.
git reset hard
Elimna los archivos del repositorio, del stage y del directorio.
git rm
Elimina los archivos del directorio pero no elimina las versiones del repositorio
git checkout
Permite cambiar entre las versiones que existen en el repositorio y permite cambiar entre las ramas existentes.
git branch
Permite crear una nueva rama. Se debe agregar el nombre de la nueva rama.
git checkout -b
Genera una nueva rama y nos mueve a ella directamente
git resert id-commit
Nos lleva a cualquier commit no importa la rama, eliminando el historial de los commit posteriores al tag seleccionado.
git reset rama-o-id-commit
Nos lleva a cualquier commit sin borrar los commit posteriores al tag seleccionado.
